# Quickstart: 번개 모임 정산기

## Project Setup

1.  **Install dependencies:**

    ```sh
    npm install
    ```

2.  **Run the development server:**

    ```sh
    npm run dev
    ```

3.  Open your browser to the provided URL (usually `http://localhost:5173`).

## Running Tests

To run unit tests, use the following command:

```sh
npm run test:unit
```
